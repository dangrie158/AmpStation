# Class D Amplifier Project

This project aims to repurpose an old Teufel Decoder Station Case by building a Class D amplifier inside it.

## Introduction

The old Teufel Decoder Station Case has been transformed into a powerful Class D amplifier in this project.

## Features

- Compact design that fits perfectly into the Teufel Decoder Station Case
- High-efficiency Class D amplifier technology
- Excellent audio quality with low distortion
- Multiple input options for versatile connectivity
- Built-in protection circuitry for safe operation
